Propiedades
de las Relaciones

Existen varias propiedades asociadas a las relaciones, las cuales son:

  • Relación Reflexiva
  • Relación Irreflexiva
  • Relación Simétrica
  • Relación Antisimétrica
  • Relación Asimétrica
  • Relación Transitiva

Sea A un conjunto y R ⊆ A × A una relación.

Relación Reflexiva

Definición

Una relación es reflexiva si todo elemento del conjunto se relaciona consigo mismo.

Condición matemática

∀a ∈ A, (a,a) ∈ R

Ejemplo

A = {1,2,3}
R = {(1,1),(2,2),(3,3)}
✔ Reflexiva

Relación Irreflexiva

Definición

Una relación es irreflexiva si ningún elemento del conjunto se relaciona consigo mismo.

Condición matemática

∀a ∈ A, (a,a) ∉ R

Ejemplo

R = {(1,2),(2,3)}
✔ Irreflexiva

Relación Simétrica

Definición

Una relación es simétrica si al existir un par, también existe su inverso.

Condición matemática

∀a,b ∈ A, (a,b) ∈ R ⇒ (b,a) ∈ R

Ejemplo

R = {(1,2),(2,1)}
✔ Simétrica

Relación Antisimétrica

Definición

Una relación es antisimétrica si no existen pares opuestos distintos.

Condición matemática

(a,b) ∈ R ∧ (b,a) ∈ R ⇒ a = b

Ejemplo

R = {(1,2),(2,3),(3,3)}
✔ Antisimétrica

Relación Asimétrica

Definición

Una relación es asimétrica si nunca ocurre el par inverso.

Condición matemática

(a,b) ∈ R ⇒ (b,a) ∉ R
(a,a) ∉ R

Ejemplo

R = {(1,2),(2,3)}
✔ Asimétrica

Relación Transitiva

Definición

Una relación es transitiva si las relaciones se encadenan.

Condición matemática

(a,b) ∈ R ∧ (b,c) ∈ R ⇒ (a,c) ∈ R

Ejemplo

R = {(1,2),(2,3),(1,3)}
✔ Transitiva
