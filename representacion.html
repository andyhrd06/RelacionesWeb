Representación de las Relaciones


El representar las relaciones de otra forma nos facilita el análisis de sus propiedades.

Una relación puede representarse de distintas formas:


Dados dos conjuntos no vacíos A y B, una relación R es un conjunto de pares ordenados en donde el primer elemento "a" está relacionado con el segundo elemento "b" por medio de cierta propiedad o característica. La relación se indica como aRb:

R = {(a, b) | a ∈ A y b ∈ B}

Ejemplo:

Sean los conjuntos:

A = {a | a ∈ Z; 10 < a < 30}
B = {b | b ∈ Z+; b < 20}

y sea R una relación de A en B, en donde el elemento a ∈ A es divisible entre 13 y b ∈ B es primo.

Como resultado se obtiene la siguiente relación:

R = {(13, 2), (13, 3), (13, 5), (13, 7), (13, 11), (13, 13), (13, 17), (13, 19), (26, 2), (26, 3), (26, 5), (26, 7), (26, 11), (26, 13), (26, 17), (26,19)}


Una tabla organiza la información en filas horizontales y columnas verticales. La primera fila o columna suele decir lo que representan los números.

Ejemplo:

La siguiente tabla muestra una relación entre el precio en dólares y la cantidad de mangos que se pueden comprar. Por ejemplo, cuando el precio es de 2 dólares, se obtienen 3 mangos. Si el precio aumenta a 4 dólares, la cantidad de mangos aumenta a 6.

Se observa que al aumentar el dinero, aumenta proporcionalmente la cantidad de mangos, manteniéndose una relación constante. Esta tabla permite visualizar de forma ordenada cómo varía una cantidad en función de la otra.


Las digráficas o diagramas gráficos son representaciones visuales de una relación entre dos conjuntos de datos usando ejes, puntos o líneas en un plano.

Las digráficas permiten:

  • Visualizar fácilmente cómo cambia una variable cuando cambia la otra.
  • Identificar si la relación es creciente, decreciente o constante.
  • Comparar datos y detectar patrones o tendencias.
  • Entender relaciones sin necesidad de leer muchos números en una tabla.

Cada círculo es un elemento y cada flecha indica una relación dirigida de un elemento a otro. Esta representación permite visualizar fácilmente cómo se conectan los elementos y comprender la relación sin listar todos los pares ordenados.


La representación matricial de una relación consiste en usar una matriz para mostrar qué elementos se relacionan entre sí. Las filas representan el elemento de salida y las columnas el elemento de llegada. Si existe relación entre dos elementos, se coloca un 1; si no existe, se coloca un 0. Esta forma de representación permite identificar relaciones de manera ordenada y facilitar su análisis.
